
Instalar:
* npm install -g @angular/cli
* npm install -g ionic

Verificar a versão: 

Angular: ng --version
Ionic: ionic --version

Criar o projeto: ionic start eshop blank

Executar o projeto: ionic serve

Build: ionic build


Criar um módulo com nome de "components": ionic generate module components

Criar um component para o módulo "components": ionic generate component components/loading
